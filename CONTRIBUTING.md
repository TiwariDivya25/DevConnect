# Contributing to DevConnect

Welcome! Follow these steps to contribute:

1. Fork the repository
2. Clone your fork locally
3. Create a feature branch
4. Make changes and commit with descriptive messages
5. Push your branch
6. Open a Pull Request referencing the related issue

### Code Style
- TypeScript with strict mode
- Tailwind CSS for styling
- Functional React components

### Issues & PRs
- Always link your PR to an issue (#issue_number)
- Label your PR as `frontend`, `backend`, or `documentation`
